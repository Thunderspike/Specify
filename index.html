table

Generally used in administrative contexts, the table widget can be more lively than people give it credit for

The Input Editor expects an object with structure { data: [ header, ...rows ] } where header is an array of the header values, and ...rows are any number of arrays representing the data you want in your generated table

The Configuration Editor could be given an object that configures features of the table. See below for an example

Tip: Type "[]" to reset to a valid input object for the Input Editor on the left, and "{}" to reset to a valid configuration for Configuration editor on the right
